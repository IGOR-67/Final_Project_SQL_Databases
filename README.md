# Итоговый проект: SQL и базы данных

Проект посвящен анализу данных о вакансиях и работодателях с использованием SQL. Работа включает несколько последовательных этапов, направленных на всестороннее исследование данных.

## Этапы проекта

### 1. Знакомство с данными
- Изучение структуры базы данных: таблицы, связи, типы данных.
- Обзор содержимого таблиц для понимания характера хранимой информации.
- Проверка целостности и качества исходных данных.

### 2. Предварительный анализ данных
- Выявление пропущенных значений и аномалий в данных.
- Оценка распределения данных по ключевым полям.
- Формирование общих статистических показателей по всему датасету.

### 3. Детальный анализ вакансий
- Исследование динамики публикации вакансий по времени.
- Анализ уровня заработных плат в разрезе должностей, регионов и опыта.
- Изучение требований к навыкам (keywords) и условиям работы.
- Сегментация вакансий по сферам деятельности и специализациям.

### 4. Анализ работодателей
- Рейтинг работодателей по количеству активных вакансий.
- Оценка средней предлагаемой зарплаты среди работодателей.
- Анализ географического распределения компаний.
- Исследование взаимосвязи между размером компании и условиями вакансий.

### 5. Предметный анализ
- Углубленное исследование по конкретным профессиональным областям (например, IT, маркетинг, финансы).
- Сравнительный анализ спроса и предложения на рынке труда.
- Формулировка выводов и рекомендаций на основе проведенного анализа.

## Используемые технологии и инструменты
- **SQL**  — выполнение запросов и анализ данных.
- **Python** (опционально, для визуализации) — построение графиков и диаграмм.
- **Jupyter Notebook** — среда для написания и выполнения кода.

## Результаты
По итогам каждого этапа формируются выводы, которые могут быть использованы для:
- Понимания текущих тенденций на рынке труда.
- Подготовки отчетности и аналитических справок.
- Поддержки принятия решений в HR-аналитике и карьерном консультировании.
- labor_market_competition.xlsx сохранён с ключевыми срезами и топами работодателей
